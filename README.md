# Kochi Lens - E-commerce SRS

## Giới thiệu
Dự án xây dựng hệ thống E-commerce bán thiết bị ngành ảnh cho Kochi Lens.  
Hệ thống cho phép khách hàng đặt hàng trực tuyến và đồng bộ dữ liệu về backend để xử lý kho và kế toán.
